# 소울북 용어 사전 (Soulbook Vocabulary)

> 소울북의 모든 UI 텍스트, 코드 주석, 문서에서는 아래의 용어를 우선적으로 사용합니다.  
> 일반적인 커머스 용어 대신 소울북만의 언어로 세계관을 유지해 주세요.

---

## 운영 원칙

> **"사용자에게 딱딱한 거래의 느낌이 아닌, 소중한 물건을 아끼는 친구에게 전달하는 느낌을 주어야 함"**

소울북은 중고 거래 플랫폼이 아닙니다.  
책을 사랑하는 사람들이 서로에게 소중한 책을 **건네는** 공간입니다.  
모든 문구, 버튼, 안내 메시지는 따뜻하고 감성적인 어조를 유지해야 하며,  
'판매', '구매', '상품' 같은 냉정한 상업적 표현은 지양합니다.

---

## 용어 대조표

| 일반 용어 | 소울북 용어 | 설명 |
|-----------|-------------|------|
| 판매자 | **옛친구** | 책을 건네주는 사람 |
| 구매자 | **새친구** | 책을 건네받는 사람 |
| 판매하기 | **건네주기** | 책을 등록하고 전달하는 행위 |
| 구매하기 | **건네받기** | 책을 발견하고 이어받는 행위 |
| 상품명 | **소울북** | 건네지는 책 한 권 |
| 가격 | **친구비** | 책과 마음을 이어주는 작은 비용 |
| 채팅 | **책 대화** | 책을 매개로 나누는 1:1 대화 |
| 리뷰 / 후기 | **책 인연 후기** | 거래 완료 후 남기는 따뜻한 소감 |

---

## 사용 예시

```
// ❌ 지양
<button>구매하기</button>
<p>판매자가 상품을 등록했습니다.</p>

// ✅ 권장
<button>건네받기</button>
<p>옛친구가 소울북을 올렸습니다.</p>
```

---

## 주의사항

- 코드 내 변수명·함수명은 영문을 사용하되, 주석과 UI 문구는 반드시 소울북 용어를 따릅니다.
- 새로운 기능을 추가할 때도 이 세계관에 맞는 용어를 먼저 정의한 후 개발을 시작합니다.
- 용어가 추가되거나 변경될 경우 이 문서를 먼저 업데이트합니다.
